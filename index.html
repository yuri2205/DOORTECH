Proyecto: Puerta Automática con Arduino

Sobre el Proyecto

Este proyecto académico consiste en el desarrollo de una puerta automática controlada por Arduino para el acceso vehicular en estacionamientos. Utiliza un sensor ultrasónico (HC-SR04) para detectar vehículos y un servomotor para abrir o cerrar la barrera automáticamente.

Objetivo General

Automatizar mediante la implementación de Arduino un sistema de puerta automática que permita detectar la presencia de vehículos y controlar el acceso a un estacionamiento de manera segura, eficiente y automatizada.

Objetivos Específicos

  • Implementar un sensor ultrasónico para detectar vehículos
  • Programar el Arduino para procesar señales y activar la apertura automática
  • Integrar un servomotor que realice el movimiento de la puerta
  • Diseñar un circuito seguro y funcional para todos los componentes
  • Evaluar el rendimiento del prototipo en pruebas de acceso vehicular
